Search a Profile Bot is an Automated Google Talk Bot that will return a wealth of information about an email id.

All you need to do is provide an Email Address and in return, it uses the RainMaker API to fetch information like:

  • Demographics like Age, Sex, Geographical location where the contact resides
  • Contact Handles for various services like Google Chat, Skype, etc.
  • Social Profiles at LinkedIn, Twitter, Facebook, Foursquare, Google Profile, Friendster, MySpace, etc.
  • Employment History.

Refer to the 2 minute guide to get started today!
